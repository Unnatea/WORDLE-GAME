Wordle Game

A simple word guessing game

About the Project

This is a simple browser based word guessing game. In this the user guesses a 5 letter words in 6 tries and the feedback is given in colours.

Idea

The aim was to build this game using only basic HTML, CSS, and JavaScript without using any external libraries.

What I Learned

I learned DOM manipulation, event handling, and how to implement game logic using JavaScript.
